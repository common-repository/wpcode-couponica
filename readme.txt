== Description ==

Using this small plugin you'll be able to create a coupon website

When your users click on coupon:

* They copy coupon code
* Automatically will be opened the couponized website (use affiliation links ;))

Works with:

* Php
* Css
* Jquery
* custom Fields

== Installation ==

Install this plugin is very simple:

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate plugin through the 'Plugins' menu in WordPress
3. Set Plugin Into your Wordpress Dashboard
3. Place:

* Shortcode [couponica]
or
* couponica function into your wordpress theme
* Custom field called "couponica_code" (coupon code)
* Custom field called "couponica_link" (website link)
